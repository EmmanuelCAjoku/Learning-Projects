# Generic Library Catalog System

## Overview
This project implements a generic library catalog system in Java using generic classes and methods. The application demonstrates flexibility and code reusability by allowing different types of library items to be managed within a single catalog structure.

## Features
- Add new library items
- Remove existing items
- Retrieve item details
- Display all catalog items
- Error handling for invalid operations
- Generic class implementation for flexibility and reusability

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Java Generics
- ArrayList Collections
- Command-Line Interface (CLI)

## Project Structure
- `LibraryItem.java` → Generic library item class
- `GenericCatalog.java` → Generic catalog management class
- `LibraryManagementSystem.java` → Main application interface

## Concepts Demonstrated
- Generic classes and methods
- Encapsulation
- Modularity
- Reusability
- Exception handling
- Collection framework usage

## How to Run
1. Compile all Java files:
```bash
javac *.java
```

2. Run the application:
```bash
java LibraryManagementSystem
```

## Sample Operations
- Add a library item
- Remove an item
- Retrieve item details
- View complete catalog
